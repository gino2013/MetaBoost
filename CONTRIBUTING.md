# Contributing to MetaBoost

All contributions are welcome — new tools, better docs, bug fixes, or usage stories.

---

## How to Contribute

1. **Fork** this repository
2. **Create a branch**: `git checkout -b feature/add-new-tool`
3. **Make your changes**
4. **Open a Pull Request** with a clear description

---

## What to Contribute

- New open-source tools for Instagram / Threads
- Improved documentation or translations
- Example scripts and use cases
- Bug fixes in example code

---

## Tool Submission Format

When adding a new tool to the README, use this format:

```markdown
| [Tool Name](github-link) | Language | ⭐ Stars | One-line description |
```

And add a section in the relevant `docs/0X-*.md` file with:
- Install steps
- Basic usage code snippet
- Link back to the tool's repo

---

## Code Style

- Python: follow PEP 8, use type hints where helpful
- JavaScript/TypeScript: use `const`, prefer async/await
- Keep examples minimal and copy-pasteable

---

## Questions?

Open a [GitHub Issue](https://github.com/gino2013/MetaBoost/issues) — all feedback welcome.
