## What is MetaBoost?

MetaBoost is a curated toolkit and integration guide for **Instagram** and **Threads** creators. It bundles the best open-source tools for scheduling, automation, analytics, and growth — all in one place.

Whether you're a solo creator or managing multiple brand accounts, MetaBoost gives you a complete toolchain: from content scheduling and automated engagement to deep analytics and Threads-specific features.

---

## Start in 3 Steps

```bash
# 1. Clone this repo
git clone https://github.com/gino2013/MetaBoost.git
cd MetaBoost

# 2. Install your chosen tool (e.g. instagrapi)
pip install instagrapi

# 3. Run an example script
python examples/basic_analytics.py
```

See [Getting Started](docs/01-getting-started.md) for full setup.

## 什麼是 MetaBoost？
